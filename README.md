# 🧩 Quiz para Programadores

## 📌 Descrição
Projeto em **HTML e CSS** que implementa um quiz interativo para testar conhecimentos de programação.  
Inclui perguntas de múltipla escolha, texto, senha, data, seleção múltipla, upload de arquivo, menu suspenso e imagens.

---

## 🚀 Etapas principais
1. Estruturação da página com HTML5.
2. Criação de perguntas interativas em diferentes formatos.
3. Implementação de tabela de pontuação com avaliações.
4. Lista de respostas corretas para conferência.
5. Formulário de feedback para os participantes.
6. Página adicional **Sobre Mim** com informações pessoais e interesses.

---

## 🛠️ Tecnologias
- HTML5  
- CSS3  

---

## 📊 Resultado
Quiz funcional e interativo que permite ao usuário testar seus conhecimentos em programação, verificar respostas corretas e enviar feedback.  
Inclui também uma página de apresentação pessoal.
